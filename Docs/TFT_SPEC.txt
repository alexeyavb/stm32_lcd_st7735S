1. Особенности

Libraries Adafruit_ST7735 или UCGlib

Режим отображения   Нормально черный ИПС
Формат дисплея   Графический 240RGB * 240 Точечно-матричный
Входные данные   Интерфейс SPI
Привод IC   ST7789VW
Размерный контур   27,78 (Ш) * 39,22 (В) * 3,0 +/-0,1 (Т) мм
Разрешение   240RGB * 240 точек
Активная область ЖК   23,4 (Ш) * 23,4 (В)
Шаг точки   0,0975 (В) x 0,0975 (В) мм
Направление просмотра   Все Посмотреть
Рабочая температура   -20 ~ 70 ° C
2. Описания штифта:
ПИН №   Символ   Описание
1   Земля   Земля
2   VCC   Мощность 3,3 V
3   СКЛ   Последовательный штифт часов.
4   ПДД   Последовательный ввод данных
5   ВИЭ   LCM Сбросить контакт
Этот вывод является входом сигнала сброса. Когда контакт низкий, инициализация микросхемы
Казнен. Держите этот штифт высоко во время нормальной работы.
6   Постоянный ток   Данные/командное управление
7   7BLKКонтрольный штифт подсветки   Backlight control pin
1,77-дюймовый ЖК-модуль TFT:
1,8-дюймовый ЖК-модуль TFT:
1,8-дюймовый последовательный порт SPI 128*160 Инструкции
Метод подключения ввода-вывода программы по умолчанию:
Sbit LCD_CS = P0 ^ 0; // Выбор микросхемы
Sbit LCD_AO = P0 ^ 2; // Переключатель данных/команд
Sbit LCD_SDA = P1 ^ 5; // Данные
Sbit LCD_SCK = P1 ^ 7; // часы
Sbit LCD_REST = P0 ^ 1; // сброс
Источник питания TFT: источник питания экрана 2,8-3,3 В; не используйте 5 В.
(Модули с печатной платой унаследовали IC регулятора 3 В, может быть Вход 5 В)
Источник питания подсветки (светодиодный контакт): максимальный источник питания подсветки составляет 3,2 В..
